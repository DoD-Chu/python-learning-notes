## pyinstaller打包
### 常用命令：pyinstaller -F -w  pyinstaller_demo.py
> -F 表示生成单个可执行文件(注意大小写) <br/>
> -w 表示去掉控制台窗口，这在GUI界面时非常有用。不过如果是命令行程序的话那就把这个选项删除吧！ <br/>
> -p 表示你自己自定义需要加载的类路径，一般情况下用不到 <br/>
> -i 表示可执行文件的图标 <br/>
